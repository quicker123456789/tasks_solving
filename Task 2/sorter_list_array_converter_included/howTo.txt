1) Исходники в папке Code. Я не стал раскидывать классы по отдельным файлам, т.к. так проще ознакомиться с кодом, не имея под рукой IDE.
2) Скомпилировання версия для тестирования в папке out.
3) Входные файлы для тестирования находятся в папке testData.
4) Для запуска программы понадобится:
4.1) Установить jre (https://www.java.com/ru/download/)
4.2) Находясь в папке out выполнить в командной строке команду вида:

путь_до_java.exe(там_где_установлена_jre) Program [параметры запуска]

например:

"D:\temp\java\bin\java.exe" Program "..\testData\in_str.txt" "..\testData\out_str.txt" -s -a
"D:\temp\java\bin\java.exe" Program "..\testData\in_str.txt" "..\testData\out_str.txt" -s -d
"D:\temp\java\bin\java.exe" Program "..\testData\in_int.txt" "..\testData\out_int.txt" -i -a
"D:\temp\java\bin\java.exe" Program "..\testData\in_int.txt" "..\testData\out_int.txt" -i -d

P.S. В условии задачи сказано, что в файлах могут содержаться любые _непробельные_ символы. Поэтому если в файле с числами будет строка, содержащая пробел, например "300 ", это будет трактоваться как ошибка исходных данных и приведет к завершению выполнения программы.
